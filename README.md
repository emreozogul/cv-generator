## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/yourusername/cv-generator.git
```

2. Install dependencies:
```bash
cd cv-generator
npm install
```

3. Run the development server:
```bash
npm run tauri dev
```

## Building

To create a production build:

```bash
npm run tauri build
```
