# CV Generator

A modern CV/Resume generator built with React, TypeScript, and Tauri. Create professional CVs with ease, featuring multiple templates and export options.

## Preview

![CV Generator Preview](./screenshots/preview.png)

## Features

- 🎨 Multiple professional templates
- 📱 Responsive design
- 💾 Local data persistence
- 📄 Export to PDF
- 🎯 Easy customization
- 🔍 Real-time preview

## Sections

- Personal Information
- Professional Summary
- Work Experience
- Education
- Skills (with grouping)
- Projects
- Custom sections

## Technologies Used

- React
- TypeScript
- Tauri
- Tailwind CSS
- Zustand (State Management)
- html2pdf.js (PDF Export)

## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/yourusername/cv-generator.git
```

2. Install dependencies:
```bash
cd cv-generator
npm install
```

3. Run the development server:
```bash
npm run tauri dev
```

## Building

To create a production build:

```bash
npm run tauri build
```
